Pill-shaped call-to-action button — use for the primary action on any section, form, or card.

```jsx
<Button variant="primary" size="lg" onClick={handleShopNow}>Shop Now</Button>
<Button variant="secondary">Our Story</Button>
<Button variant="text">View all reviews</Button>
```

Variants: `primary` (solid green, lifts + darkens on hover), `secondary` (outlined, inverts to solid ink on hover), `text` (underlined link, used for tertiary actions). Sizes: `sm` / `md` / `lg`. Set `disabled` to dim to 50% opacity and block interaction.

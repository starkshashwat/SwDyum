Icon + title(+subtitle) inline chip — trust bar scroller, PDP benefit row.

```jsx
<Chip icon={<LeafIcon />} title="100% Organic" />
<Chip icon={<SunIcon />} title="Sun-Cured" subtitle="Authentic taste" />
```

Single-line (`title` only) for the trust bar; two-line (`title` + `subtitle`) for PDP benefit callouts.
